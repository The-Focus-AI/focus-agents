# Template Selection Guide

Choose the appropriate template based on your research needs.

## Quick Reference

| Research Need | Template | Time | File |
|---------------|----------|------|------|
| Comprehensive topic analysis | Standard Academic | 30-60 min | `templates/academic.md` |
| Business decision support | Executive Summary | 15-30 min | `templates/executive-summary.md` |
| Evaluating options | Comparative Analysis | 20-40 min | `templates/comparative-analysis.md` |
| Academic survey | Literature Review | 45-90 min | `templates/literature-review.md` |
| Rapid answer needed | Quick Report | 10-15 min | `templates/quick-report.md` |
| Library/tool selection | Technical Implementation | 30-45 min | `templates/technical-implementation.md` |

## When to Use Each Template

### Standard Academic
- In-depth topic research
- Creating knowledge base articles
- Formal documentation
- When thoroughness matters more than speed

### Executive Summary
- Briefing stakeholders
- Quick business decisions
- When actionable recommendations are the priority
- Time-constrained but needs to be presentable

### Comparative Analysis
- Choosing between technologies
- Vendor/tool evaluation
- Pros/cons analysis
- Decision matrices needed

### Literature Review
- Academic or scholarly research
- State-of-the-art surveys
- Historical context important
- Formal methodology required

### Quick Report
- Time-sensitive requests
- Preliminary research
- Simple factual questions
- Exploratory research before deeper dive

### Technical Implementation
- Library or framework selection
- Tool recommendations for developers
- Implementation guides with code examples
- When setup instructions and patterns are needed
- Decision rationale documentation (why this choice, alternatives considered)
- Future-proofing: helps revisit decisions later with full context
